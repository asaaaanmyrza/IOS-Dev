soz of the day
==============

soz of the day is an educational iOS application designed to help users learn the Kazakh language in an engaging and practical way. The app focuses on daily exposure to vocabulary, phrases, and curated learning resources, with an emphasis on Latin Kazakh.

--------------------------------------------------
FEATURES OVERVIEW
--------------------------------------------------

The application consists of three main interfaces:

--------------------------------------------------
1. Soz of the Day
--------------------------------------------------

This screen displays a randomly selected Kazakh word from the dictionary.

Each word card includes:
- The Kazakh word
- An English definition
- A second card with three example sentences showing how the word is used in everyday speech

The word updates automatically every 30 seconds to encourage active and continuous learning rather than memorizing a single word per day.

--------------------------------------------------
2. Kokjiek
--------------------------------------------------

This section provides a curated list of beginner-friendly YouTube resources for learning Kazakh.

For each video, the user can see:
- Video thumbnail
- Title
- Description

By tapping on a video, the user is redirected directly to YouTube via the video URL.

--------------------------------------------------
3. Olen of the Day
--------------------------------------------------

This screen presents meaningful Kazakh phrases sourced from interviews, song lyrics, literature, and other cultural materials.

Each entry includes:
- The original Kazakh phrase
- English translation
- Explanation or definition
- Author’s name
- Author’s image

They are also update automatically every 30 seconds.

--------------------------------------------------
PURPOSE
--------------------------------------------------

The goal of soz of the day is to make learning Kazakh more accessible, modern, and practical by combining vocabulary, real-life phrases, and multimedia learning resources in one place.
